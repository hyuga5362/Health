# 健康維持のためのアプリ
**calendar-app - 健康管理を簡単に。

## 概要
calendar-appは少し手間のかかる健康管理を簡単に可視化でき、めんどくさがりな方でも簡単に習慣化できるWEBアプリです。
開発途中にはなりますが簡単な動作確認が可能になっています、ぜひお試しください。
